# Проект Блог
### Описание: проект на FastAPI. Позволяет смотреть блог с различными постами,создавать новых пользователей, новые посты, а также редактировать их.
## Структура проекта:
* app/
    * main.py
    * models.py
    * storage.py
    * views.py
    * data/
        * store.json
    * routers/
        * posts.py
        * users.py
    * static/
        * css/
            * base.css
            * home.css
            * post_form.css
            * post_view.css
            * user_form.css
    * templates/
        * base.html
        * home.html
        * post_form.html
        * post_view.html
        * user_form.html
* requirements.txt
* README.md
* .gitignore
